AI Prompts for QA Work

A collection of reusable prompts for working with AI tools such as ChatGPT or Claude.

The prompts are focused on practical Quality Assurance tasks, including API testing, test documentation, test automation, requirements analysis, and AI evaluation.

Purpose: 
The purpose of this repository is to collect, organize, and improve prompts that can be used in everyday QA work.

These prompts can help with tasks such as:
* generating API test cases
* creating test plans and checklists
* preparing bug reports
* analyzing requirements
* reviewing Python and test automation code
* creating test reports
* generating AI evaluation scenarios
* explaining technical concepts and code

AI-generated results should always be reviewed and adjusted before they are used in a real project.
